# Safety-Driver-Application
GPS를 이용하여 속도 및 방위각 데이터를 (교통안전공단 위험운전 행동분석 기준 및 정의 기반) 안전 운전 판단 알고리즘으로 분석하여 위험 운전을 캐치, 운전자에게 위험 운전 습관 데이터 분석 및 비교 해주는  Application
